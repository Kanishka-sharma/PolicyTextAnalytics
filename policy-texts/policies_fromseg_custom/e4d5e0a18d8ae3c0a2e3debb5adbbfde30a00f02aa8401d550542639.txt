What is a cookie?
A cookie is a small text file that is placed on your hard drive by a web page server.
==========
Cookies contain information that can later be read by a web server in the domain that issued the cookie to you.
Some of the cookies will only be used if you use certain features or select certain preferences, and some cookies will always be used.
You can find out more about each cookie by viewing our current cookie list here.
We update this list quarterly, so there may be additional cookies that are not yet listed.
