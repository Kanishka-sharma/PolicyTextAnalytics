Cookies

Cookies are commonly used by many web sites to facilitate the site visit.
A cookie is a data file that is transferred to your browser by a web server.
Only the web server sending the cookie has access to it.Our studies require that your browser be set to accept cookies only to ensure that you are not solicited to reply to our questionnaire more than once while the current study is running.
After that it automatically expires and our web server can no longer access it.On initial setup, most web browsers are set to accept cookies.
On most browsers, you can set an option for the browser to notify you when a cookie is being sent with the option of accepting or rejecting the cookie.
